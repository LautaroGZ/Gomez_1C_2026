Detailed Description

Medidor de distancia por ultrasonido con visualización en LCD.

Sistema de medición de distancia basado en sensor ultrasónico HC-SR04 con control de LEDs y display LCD para visualización de resultados.

◆ TareaMedicion()

void TareaMedicion ( void * pvParameter)

Tarea encargada de medir la distancia y actualizar salidas.

Parameters
[in]pvParameterParámetro de la tarea (no utilizado)
Returns
void
  • Realiza la lectura del sensor ultrasónico HC-SR04
  • Actualiza el valor en el display LCD (si no está en hold)
  • Controla el encendido de LEDs según rangos de distancia:
    • < 10 cm: Todos los LEDs apagados
    • 10-20 cm: LED_1 encendido
    • 20-30 cm: LED_1 y LED_2 encendidos
    • > 30 cm: Todos los LEDs encendidos
  • Período de ejecución: 1000 ms

Lectura del sensor ultrasónico HC-SR04 Obtiene la distancia en centímetros

Actualización del display LCD Solo se actualiza si no está en modo hold

Control de LEDs según rangos de distancia Los LEDs se encienden progresivamente conforme aumenta la distancia

Sistema pausado Apaga todos los LEDs y desactiva el display

Espera antes del siguiente ciclo de medición Período: 1000 ms

◆ TareaTeclas()

void TareaTeclas ( void * pvParameter)

Tarea encargada de leer el estado de las teclas por polling.

Parameters
[in]pvParameterParámetro de la tarea (no utilizado)
Returns
void
  • Lee el estado de las teclas cada 50 ms
  • TEC1: Alterna entre pausar/reanudar la medición (medir_activa)
  • TEC2: Alterna entre modo hold/normal del display (hold_activo)
  • Período de ejecución: 50 ms

Variable local para almacenar el estado de las teclas

Cada bit representa el estado de una tecla específica
