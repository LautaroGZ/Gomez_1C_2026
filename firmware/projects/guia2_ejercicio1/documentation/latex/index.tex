\chapter{Firmware para placa EDU-\/\+ESP que mide distancia utilizando un sensor ultrasónico HC-\/\+SR04 }
\hypertarget{index}{}\label{index}\index{Firmware para placa EDU-\/ESP que mide distancia utilizando un sensor ultrasónico HC-\/SR04@{Firmware para placa EDU-\/ESP que mide distancia utilizando un sensor ultrasónico HC-\/SR04}}
\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{General Description}{General Description}}\label{index_genDesc}
Firmware para la placa EDU-\/\+ESP que mide distancia utilizando un sensor ultrasónico HC-\/\+SR04. Muestra la distancia en un display LCD y controla un arreglo de LEDs según rangos de distancia. Permite pausar/\+reanudar la medición y mantener (hold) el valor en pantalla usando pulsadores.\hypertarget{index_hardConn}{}\doxysection{\texorpdfstring{Hardware Connection}{Hardware Connection}}\label{index_hardConn}
